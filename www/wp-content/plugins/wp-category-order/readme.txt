== Description ==

This plugins allows you to display checked categories in the hierarchical order. I.e., the category list will be displayed in the order and will not move selected category to the top.
It helps in managing the order of parent and child categories so that on edit post User can easily check and follow the hierarchy of the category.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Check on the Back-end, you will get category in hierarchical order.
